# Гайд по использованию сайта


## Вход в аккаунт

- Для входа используйте следующие данные:
  - **Логин:** `root`
  - **Пароль:** `123`

![Скриншот 1: Форма входа](https://res.cloudinary.com/dl03vhi5u/image/upload/v1764502482/01_lbzaa0.jpg)

## Панель администратора

  
- После входа вы увидите панель администратора.

![Скриншот 2: Панель администратора](https://res.cloudinary.com/dl03vhi5u/image/upload/v1764502483/02_bqwqvk.jpg)
  
- Для создания нового проекта нажмите кнопку **"Создать проект"**.
- Вы будете перенаправлены на страницу создания проекта.

![Скриншот 3: Страница создания проекта](https://res.cloudinary.com/dl03vhi5u/image/upload/v1764502482/03_ixrqku.jpg)
## Вставка виджета на сайт

- После создания проекта появится код для вставки виджета.
- **Не используйте кнопку копировать!** Скопируйте код вручную и вставьте его на любой сайт **перед закрывающим тегом `</body>`**.

![Скриншот 4: Код для вставки виджета](https://res.cloudinary.com/dl03vhi5u/image/upload/v1764502482/04_hgolhg.jpg)
## Фильтрация заявок

- В панели можно фильтровать заявки разными способами: по статусу, исполнителю, дате и т.д.

![Скриншот 5: Фильтрация заявок](https://res.cloudinary.com/dl03vhi5u/image/upload/v1764502482/05_zk0aup.jpg)
## Просмотр заявки

- Нажмите на заявку, чтобы открыть подробности. Заявка состоит из следующих частей:
  - Данные пользователя
  - Описание проблемы
  - Статус
  - Исполнитель
  - Дата создания и обновления

![Скриншот 6: Детали заявки](https://res.cloudinary.com/dl03vhi5u/image/upload/v1764502482/06_lbjz7f.jpg)
## Назначение исполнителя и изменение статуса

- Внутри заявки можно назначить исполнителя и изменить статус заявки.

![Скриншот 7: Назначение исполнителя и статуса](https://res.cloudinary.com/dl03vhi5u/image/upload/v1764502481/07_xx5zqu.jpg)

## Django Admin

- Для управления пользователями и другими сущностями перейдите в Django Admin.
- Обычно это доступно по адресу: `https://api.neurocity.app/admin/`
- Здесь вы можете создать нового пользователя или изменить существующих.


## Быстрый старт

1. Войдите в аккаунт с логином `root` и паролем `123`.
2. Ознакомьтесь с панелью администратора.
3. Создайте новый проект.
4. При необходимости перейдите в Django Admin для управления пользователями.
5. Добавьте скриншоты в папку `screenshots` и вставьте их в этот файл по примеру выше.
